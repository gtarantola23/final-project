A Comparison of 3 Clustering Algorithms

For this page, we aimed to create a useful visualization to clearly show the differences in clustering algorithms. For K-means and K-means ++, the first initial centroid is the same, and then the rest are choosen by their respective initialization methods. All algorithms use the same set of data points, that is the add data button will add the same points to all three svg boxes, and when you click to add a point in one of the boxes, the point will appear in the same spot in the other two.
