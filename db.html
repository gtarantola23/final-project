DBSCAN Clustering

The major deficit to the K-means and K-means ++ Clustering algorithms is that loosely related observations may be clustered together. Every data point will be clustered eventually, even if the points are far apart.

The DBSCAN algorithm provides a density-based alternative based on the idea that a cluster is a region of high point density. The user specifies Min Neighbors and Radius that refer to the mimimum number of neighboring points that must be present within a radius for a set of points to be clustered together. The steps of the algorithm are completed as follows.

  1. Randomly pick a point in the data
  2. Assign this point to a cluster if there are at least Min Neighbors within a Radius to the point
  3. Repeat steps 1-2 recursively by repeating the neighborhood calculation for each neighboring point
