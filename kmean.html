K-Means Clustering

K-Means Clutering is one of the most common clustering algorithms. Before the start, the user must specify k that refers to the number of clusters. Given that data is present, the steps to the algorithm are completed as follows.

  1. Randomly choose/initialize k centroids
  2. Assign each point in the dataset to its nearest centroid
  3. Calculate the new mean of each centroid, using the points that were classified into its group
  4. Repeat steps 2-3 until the classified values in each centroid remain unchanged

To use the interactive visualization tool in this website, you must first add data points. You can choose the number of data points to add and the distribution of the added data. In addition, you can manually add data points if you wish. Once you have added all the data that you would like, you can either step through or animate the algorithm after specifying the number of clusters. Once you are finished, you can reset and start again.
